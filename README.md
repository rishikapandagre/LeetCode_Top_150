# LeetCode Top 150 Solutions

This repository contains my solutions to the **Top 150 LeetCode problems**, a curated set of questions frequently asked in technical interviews. The goal is to build strong problem-solving skills, improve algorithmic thinking, and maintain clean, readable implementations.

---

## Approach

For each problem:

* Focus on **understanding patterns**, not just solving
* Prefer **optimal time and space complexity**
* Use clean and consistent coding style
* Add comments where logic is non-trivial
